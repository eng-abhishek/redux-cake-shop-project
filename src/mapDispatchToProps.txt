‚úÖ What is mapDispatchToProps?

mapDispatchToProps is a function that lets your React component send actions to the Redux store.

üëâ It basically provides functions in your component that can dispatch Redux actions.

üß† Syntax
const mapDispatchToProps = (dispatch) => {
  return {
    someFunction: () => dispatch(someAction())
  };
};


Then connect:

export default connect(mapStateToProps, mapDispatchToProps)(MyComponent);

üß© Example (Cake Example)

Suppose you have an action:

// cakeActions.js
export const buyCake = () => {
  return {
    type: 'BUY_CAKE'
  }
}


Component:

import React from "react";
import { connect } from "react-redux";
import { buyCake } from "../redux/cake/cakeActions";

function CakeContainer(props) {
  return (
    <div>
      <h2>Number of cakes: {props.numOfCakes}</h2>
      <button onClick={props.buyCake}>Buy Cake</button>
    </div>
  );
}

üëâ Using mapDispatchToProps
const mapStateToProps = (state) => {
  return {
    numOfCakes: state.cake.numberOfCakes
  };
};

const mapDispatchToProps = (dispatch) => {
  return {
    buyCake: () => dispatch(buyCake())
  };
};

export default connect(mapStateToProps, mapDispatchToProps)(CakeContainer);

üìå What happens?
Without Redux

You just call a function.

With mapDispatchToProps

The function you call (props.buyCake()) will dispatch the action ‚Üí reducer ‚Üí update store ‚Üí component re-renders.

üßæ Shortcut Version

React-Redux allows shorthand if your action creators return objects.

Instead of:

const mapDispatchToProps = dispatch => {
  return {
    buyCake: () => dispatch(buyCake())
  }
}


You can do:

export default connect(
  mapStateToProps,
  { buyCake }
)(CakeContainer);


React-Redux automatically wraps buyCake in dispatch.

üí° When do we use mapDispatchToProps?

‚úî When you want to call Redux actions from your component
‚úî When you use the connect() API (not hooks)
‚úî When your component needs to update the Redux state

# Question:: if i am using redux-hooks then how can be short above example ???

‚úÖ Short version using Redux Hooks (useSelector + useDispatch)

Here is the same example using hooks:

‚úî Component (very short)
import React from "react";
import { useSelector, useDispatch } from "react-redux";
import { buyCake } from "../redux/cake/cakeActions";

function CakeContainer() {
  const numOfCakes = useSelector((state) => state.cake.numberOfCakes);
  const dispatch = useDispatch();

  return (
    <div>
      <h2>Number of cakes: {numOfCakes}</h2>
      <button onClick={() => dispatch(buyCake())}>Buy Cake</button>
    </div>
  );
}

export default CakeContainer;

mapDispatchToProps

connect()

With hooks

useSelector replaces mapStateToProps

useDispatch replaces mapDispatchToProps

No connect() needed

Code becomes very short and easy
